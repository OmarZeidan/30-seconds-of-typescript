Returns `true` if at least one of the arguments is `true`, `false` otherwise.

Use the logical or (`||`) operator on the two given values.

```js
const or = (a, b) => a || b;
```

```js
or(true, true); // true
or(true, false); // true
or(false, false); // false
```
