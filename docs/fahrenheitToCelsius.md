Converts Fahrenheit to Celsius.

Follows the conversion formula `C = (F - 32) * 5/9`.

```js
const fahrenheitToCelsius = (degrees) => ((degrees - 32) * 5) / 9;
```

```js
fahrenheitToCelsius(32); // 0
```
